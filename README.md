# 🌍 GéoQuiz — Monde & Algérie

Quiz de géographie mondiale et algérienne, disponible en **français**, **arabe** et **anglais**.

## 🎮 Modes de jeu
- 🏛️ **Capitales** — Trouve la capitale de chaque pays
- 🚩 **Drapeaux** — Identifie le pays par son drapeau
- 🇩🇿 **Wilayas** — Trouve la wilaya algérienne par son numéro
- 🎲 **Mix** — Mélange des 3 modes

## 📱 Installation (PWA)
Ce site est une **Progressive Web App** — tu peux l'installer directement sur ton téléphone sans passer par le Play Store !

1. Ouvre le site dans Chrome (Android) ou Safari (iPhone)
2. Une bannière apparaît : **"Installer GéoQuiz"**
3. Clique **Installer** → l'icône apparaît sur ton écran d'accueil !

## 🚀 Comment lancer en local
```bash
# Clone le repo
git clone https://github.com/Sihamboudj/geoquiz.git

# Ouvre simplement index.html dans ton navigateur
# Ou utilise Live Server sur VS Code
```

## 🏆 Classement
Le classement est sauvegardé localement. La version Firebase (classement mondial en ligne) arrive bientôt !
