## 📱 Installer l'application

GéoQuiz est une **Progressive Web App (PWA)** — installe-la sur ton téléphone sans passer par le Play Store !

**Sur Android :**
1. Ouvre le lien dans **Google Chrome**
2. Une bannière apparaît → clique **"Installer"**
3. L'icône 🌍 apparaît sur ton écran d'accueil ✅

**Sur iPhone :**
1. Ouvre le lien dans **Safari**
2. Clique **Partager** → **"Sur l'écran d'accueil"** ✅

## 🎯 Modes de jeu

| Mode | Description |
|------|-------------|
| 🏛️ **Capitales** | Trouve la capitale de chaque pays |
| 🚩 **Drapeaux** | Identifie le pays par son drapeau |
| 🇩🇿 **Wilayas** | Trouve la wilaya par son numéro (#16 = Alger) |
| 🎲 **Mix** | Mélange des 3 modes |
